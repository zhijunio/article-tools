# Markdown 排版器完全指南

欢迎使用这款专为 **微信公众号** 设计的 Markdown 编辑器。左侧写作、右侧预览，一键复制到公众号后台。

## 一、核心功能

### 1. 复制到公众号 ⭐

点击 **「复制到公众号」** 按钮，一键完成所有处理：

**第一步：链接转换**

- 所有链接被替换为纯文本
- 格式：`文案 https://url`，避免公众号过滤外链

**第二步：样式内联**

- 所有 CSS 样式转为 `style="..."` 内联属性
- 移除公众号不支持的 CSS 特性

**第三步：列表兼容**

- 自定义列表符号（● ► ✓ 等）保留
- 有序列表序号自动对齐

**第四步：剪贴板写入**

- 同时写入 `text/html`（富文本）和 `text/plain`（纯文本）
- 公众号后台粘贴时自动识别富文本格式

### 2. 多主题样式

> 内置多组主题，涵盖素雅、科技、国风、极简等方向；切换主题即可预览全文效果。

- **预设主题**：暖棕书卷、黑白极简、琥珀橙调、科技蓝调、薄荷青绿、酒红复古
- **自定义主题**：调整样式后保存为新主题，支持导出/导入 JSON
- **随机换色**：一键随机生成配色方案
- **随机风格**：随机各元素的样式预设

### 3. 实时预览

- **左右分栏**：左侧编辑，右侧实时预览
- **同步滚动**：编辑区与预览区滚动位置自动同步
- **手机模拟**：预览区模拟手机显示效果（375px 宽度）

### 4. 智能粘贴

- **HTML → Markdown**：自动转换网页格式为 Markdown 语法
- **IDE 格式化识别**：识别 IDE 复制的代码，保持原有格式
- **Markdown 检测**：自动识别已是 Markdown 的内容，避免重复转换

### 5. 数据持久化

- **自动保存**：编辑内容实时保存到 localStorage
- **刷新恢复**：页面刷新后内容不丢失

### 6. 字数统计

- **字数统计**：动态计算当前字符数

## 二、Markdown 语法

### 1. 标题

支持四级标题，使用 `# ` 符号数量区分层级：

```markdown
# 一级标题

## 二级标题

### 三级标题

#### 四级标题
```

# 一级标题演示

## 二级标题演示

### 三级标题演示

#### 四级标题演示

### 2. 正文与强调

段落是文章的基本组成部分。在 Markdown 中，空一行即可创建新段落。

正文支持 **加粗**、*斜体*、`行内代码` 等格式。

### 3. 加粗、斜体与删除线

**加粗文本**：使用双星号包裹，例如 `**重要内容**`

*斜体文本*：使用单星号包裹，例如 `*强调内容*`

***加粗斜体***：使用三星号包裹，例如 `***特别强调***`

~~删除文本~~：使用双波浪线包裹，例如 `~~删除内容~~`

### 4. 行内代码

在段落中可以使用反引号包裹行内代码，例如 `console.log('hello')` 或 `npm install`。

### 5. 引用

使用 `>` 符号创建引用块：

> 基础引用：左侧写 `>`，右侧即时预览。

> 多段引用：
>
> 第一段内容。
>
> 第二段内容，空一行即可分段。

### 6. 代码块

使用三个反引号包裹代码块。内置 **highlight.js** 语法高亮引擎，提供专业级的代码展示效果：

- **多语言支持**：JavaScript、Python、Java、Go、CSS、HTML 等主流语言
- **自动检测**：未指定语言时自动识别代码类型
- **自动换行**：长代码行自动折行，避免横向滚动
- **样式保真**：复制到公众号后仍保持高亮效果

### 7. 列表

**无序列表**：使用 `-`、`+` 或 `*`

- 第一项
- 第二项
- 第三项

**有序列表**：使用数字加点

1. 第一步：准备素材
2. 第二步：编辑内容
3. 第三步：调整样式

**嵌套列表**：

- 一级列表项
    - 二级列表项 A
    - 二级列表项 B
- 另一一级列表项
