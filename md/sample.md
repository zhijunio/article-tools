# Markdown 排版器 · 示例长文

欢迎使用本工具：左侧写 **Markdown**、右侧即时预览，样式通过 **内联 `style`** 输出，适合粘贴到 **微信公众号** 等富文本环境。

## 一、你会用到的能力

### 1. 一键「复制到公众号」

- 将链接转为 **「文案 + 空格 + URL」** 的纯文本，降低外链被拦的概率  
- 样式全部 **内联**，去掉后台不吃的选择器与部分特性  
- 列表符号（如 ●、✓ 等）在剪贴板里 **尽量保留观感**  
- 同时提供富文本与纯文本，粘贴时由编辑器选用

### 2. 主题与样式

- **六种内置主题**（与说明图一致）：**优雅蓝**（默认）、**清新绿**、**温暖橙**、**深邃紫**、**经典红**、**极简黑** —— 在顶部 **主题** 下拉里切换  
- **自定义主题**：调好侧栏后保存、可 **导出 / 导入 JSON** 备份  
- **随机色**：主色与部分模块配色一键换新  
- **随机风格**：各元素（标题、引用、代码、列表等）的 **样式预设** 随机组合

### 3. 编辑与预览

- **左右分栏**，右侧可看作手机版心宽度（受 **内容最大宽度** 控制）  
- **编辑区顶栏** 可点选格式：加粗/斜体/行内代码/链接、代码块、引用、**水平线**、**三列表格**、标题 H1～H4、缩进等（与快捷键同一套行为）  
- 无修饰键 **Tab / Shift+Tab** 可 **增加 / 减少** 行首缩进（与顶栏缩进一致）  
- **滚动同步** 可在顶栏关闭，避免长文时互相拉扯  
- 需要看 **HTML 源码** 时，用对应按钮单独复制

### 4. 版式上的默认约定（与侧栏说明一致）

- **正文、列表内文**：固定为 **黑色**（#000），不随主色跑  
- **加粗颜色**：与 **全局 → 主色调** 一致  
- **引用块**：内文 **黑字、白底**；左边线、引号等 **装饰色与主色调一致**  
- **行内代码**：如 `console.log('hello')` 会原样显示引号，不会出现乱码式实体  
- **链接**：单实线下划线，不再提供「双下划线」样式

### 强调与删除

正文支持 **加粗**、*斜体*、`行内 code`，以及 ***加粗斜体*** 与 ~~删除线~~。常见写法：加粗为两侧双星号、斜体为两侧单星号（与标准 Markdown 一致）。

### 引用

> 引用里永远是 **黑字白底**；要换主色，改 **全局 → 主色调**，别指望单独给引用换「文字色 / 底纹」。

> 多段时中间 **空一行** 再继续写 `>` 即可。  
> 第二段可以很长，用来试折叠与行距。

### 代码

行内：`npm run build`、`` `a < b` ``（尖括号会正确转义显示）。

块级高亮由 highlight.js 处理，例如：

```javascript
// 行内可含单引号: console.log('ok')
const msg = "hello";
console.log(msg);
```

```python
def hello(name: str) -> str:
    return f"Hi, {name}!"
```
