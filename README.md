# Article Tools

一套本地可用的在线工具集（封面图、二维码等），可部署在 GitHub Pages 使用。

**在线预览：** https://chensoul.github.io/article-tools/

## 工具列表

- **首页** (`index.html`) — 工具入口与导航
- **封面生成** (`cover.html`) — 生成封面图，支持配色、比例、装饰、字体（Noto / MiSans / JetBrains Mono / 系统黑体）与本地保存参数
- **二维码** (`qrcode.html`) — 生成二维码，支持 Logo 等选项

## 使用方式

直接用浏览器打开对应 HTML 文件即可，无需安装任何依赖。
