# Article Tools

一套本地可用的在线工具集（封面图、Markdown 排版器、二维码等），可部署在 GitHub Pages 使用。

**在线预览：** https://zhijunio.github.io/article-tools/

## 工具列表

- **封面生成** (`cover`) — 生成封面图，支持配色、比例、装饰、字体（Noto / MiSans / JetBrains Mono / 系统黑体）与本地保存参数；脚本与字体通过 jsDelivr 加载
- **Markdown 排版器** (`editor`) — 公众号 Markdown 排版
- **二维码** (`qrcode`) — 生成与解析二维码

## 使用方式

直接用浏览器打开对应 HTML 即可；第三方库从 CDN 拉取，需联网（或使用本地 HTTP 服务以便部分环境正常加载）。
